Making Addons for SturoVoxel

Create powerful sandboxed addons using JavaScript. SturoVoxel addons are translated into gdjs (GDevelop JavaScript) while staying safe and optimized for gameplay.

Explore API View Examples 
if (!API.variables("modExampleMod123).init){
  API.print("Hello SturoVoxel!");
  API.variables("modExampleMod123).init = true
  const loader = API.GLTFloader();
  
  loader.load("mob.glb", (gltf) => {
    API.scene3D().add(gltf.scene);
  });
}

Features

The addon system provides tools for gameplay scripting, 3D rendering, world interaction, player controls, and more.

Sandboxed Addons

Addons run inside a protected environment so they cannot break the game.

JavaScript Based

Write addons using JavaScript syntax translated into gdjs.

3D Support

Access Three.js loaders, cameras, scenes, and raycasters.

Player Input

Detect keyboard and mouse events directly from your addon.

Addon API

Core functions available inside the addon sandbox.

API Documentation

API.print(value)

Prints a value to the in-game console.

API.variables(name)

Access sandbox variables for persistent addon storage.

API.getBlockInSight()

Returns the block currently targeted by the player.

API.player()

Returns the player object.

API.keyPressed(key)

Checks if a key is currently held down.

API.onKeyPressed(key)

Detects a single key press event.

API.GLTFloader()

Creates a GLTF model loader for loading 3D assets.

API.scene3D()

Returns the main Three.js scene used by the game.

API.camera()

Returns the current Three.js camera.

const path = API.getPath()
const pd = API.gethPathDelimiter()

  API.scene3D().add(gltf.scene);
});
